# Frequently Asked Questions

### How do I contribute to the documentation?

1. Go to https://github.com/velodrome-finance/docs/tree/main/content
2. Find the relevant file, if unsure where to contribute, select the `faq.mdx` (this) file
3. Find the edit button and click on it (you will need a Github account)
4. Make the necessary edits and next click on the _Commit changes..._
5. In the next screen, confirm the commit changes.
6. Done!
